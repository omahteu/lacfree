Lacfree Bot - Descrição

Bem-vindo ao Bot Lacfree! Este bot foi criado para ajudar intolerantes à lactose a calcular a quantidade ideal de UI's necessárias com base nos alimentos consumidos. Informe o alimento e a quantidade de lactose presente, e o bot orientará a dose recomendada para facilitar a digestão e reduzir sintomas de intolerância. Prático, rápido e sempre disponível para cuidar do seu bem-estar alimentar!

Cronograma de Desenvolvimento (1 Semana)

  • Dia 1: Planejamento e Configuração Inicial
  • Dia 2: Configuração da API do Bot no Telegram
  • Dia 3: Implementação do Sistema de Entrada de Dados Alimentares
  • Dia 4: Desenvolvimento do Cálculo de UI’s
  • Dia 5: Respostas e Formatação de Resultados
  • Dia 6: Testes e Ajustes Finais
  • Dia 7: Finalização e Lançamento

Bibliotecas Necessárias

  • python-telegram-bot - Para interação com o Telegram Bot API
  • pandas - Para manipulação de dados
  • sqlite3 - Para armazenar a base de dados local
  • math ou numpy - Para cálculos das UI’s com precisão
